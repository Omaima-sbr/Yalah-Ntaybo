# Yalah Ntaybo - Moroccan Culinary Social Network

## Description
Yalah Ntaybo (Let's cook in darija) is a web platform fully dedicated to Moroccan cuisine.  
This web application allows users to share, discover, and rate authentic Moroccan recipes in a culturally adapted environment.

## Objective
Create a unique digital space where the community can exchange around Moroccan gastronomy with a touch of authenticity (darija, culinary traditions).
